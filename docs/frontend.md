#### FRONTEND: WEB/PWA kudab.ru

---

#### ОБЩЕЕ ОПИСАНИЕ

Frontend kudab.ru — это современный сайт с поддержкой PWA, SSR (на Nuxt.js), оптимизированный под мобильные и десктопные браузеры.

- Полная функциональность без отдельного мобильного приложения.
- SSR для быстрой загрузки, SEO и social-preview.
- Сценарии “Пойду”, “Избранное”, экспорт событий в календарь, ручное добавление событий для организаторов.
- Глубокая интеграция с backend API и Telegram-ботом.

---

#### СТЕК

- **Фреймворк:** Nuxt.js 3.x (SSR + SPA + PWA)
- **Язык:** JavaScript/TypeScript
- **UI:** TailwindCSS, shadcn/ui, кастомные компоненты
- **PWA:** оффлайн-режим, push-уведомления, иконка, manifest.json

---

#### ОСНОВНЫЕ РАЗДЕЛЫ САЙТА

- **Главная (афиша):** подборка ближайших событий с фильтрами (дата, интересы, город)
- **Мои события:** “Пойду”, “Избранное” (личный кабинет)
- **Сообщества:** каталог групп и организаторов
- **Добавить событие:** форма для организаторов (ручное добавление, через модерацию)
- **Профиль:** настройка аккаунта, интеграция с Telegram
- **Информация:** разделы помощи, правила, поддержка

---

#### СЦЕНАРИИ ПОЛЬЗОВАТЕЛЯ

- Просмотр событий, фильтрация по интересам и дате
- Быстрое действие “Пойду” (RSVP) — появление события в личном списке
- “Избранное” — персональная подборка для напоминаний и быстрого доступа
- Экспорт события в календарь (Google, Яндекс, iCal) одной кнопкой
- Подписка на push-уведомления (через PWA)
- Добавление событий организаторами вручную (форма + подтверждение)

---

#### ИНТЕГРАЦИЯ С BACKEND И API

- Все данные и действия получаются/отправляются через REST API (kudab-api)
- Авторизация — JWT или через Telegram OAuth (одноразовый код)
- Для действий “Пойду”, “Избранное”, ручных событий — защищённые POST-запросы
- Для публичной афиши — кэшируемый GET-запрос

---

#### UI-КОМПОНЕНТЫ И ВЗАИМОДЕЙСТВИЕ

- **Карточка события:** кнопки “Пойду”, “Избранное”, “В календарь”
- **Форма добавления:** поддержка drag’n’drop для вложений, валидация дат/геолокации
- **Профиль:** интеграция с Telegram, отображение привязки

---

#### ОПЫТ ПОЛЬЗОВАТЕЛЯ (UX)

- Мгновенный отклик на действия (Toasts/snackbar на успех/ошибку)
- Уведомления о новых событиях (web-push)
- Кнопка “Добавить на главный экран” (PWA install)
- SEO- и OpenGraph-мета для лучшего отображения в Telegram, VK
