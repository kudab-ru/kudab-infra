#### BOT: TELEGRAM-БОТ KUDAB.RU

---

#### ОБЩЕЕ ОПИСАНИЕ

Telegram-бот kudab.ru — ассистент для пользователей и организаторов:
- Получение актуальных событий по интересам.
- Быстрое действие “Пойду”/“Добавить в избранное”.
- Экспорт событий в Google/Yandex/Apple Календарь (через ссылку или файл).
- Поддержка уведомлений и рассылки по подписанным каналам.
- Возможность добавления собственного события организатором (с последующей модерацией).

---

#### СТЕК

- **Язык:** Python 3.11+
- **Фреймворк:** aiogram 3.x
- **Транспорт:** REST API (через backend Laravel)
- **Данные:** Привязка telegram_users к users платформы.

---

#### СЦЕНАРИИ РАБОТЫ

- **Просмотр событий:**  
  Пользователь получает список событий с кнопками “Пойду”, “В избранное”, “В календарь”.  
  При нажатии на “Пойду” — происходит RSVP, событие появляется в разделе “Я иду”, приходит напоминание.
  При нажатии на “В избранное” — добавляет в персональный список, можно получить через /favorite.
  При нажатии на “В календарь” — бот присылает ссылку/файл для добавления события в календарь.

- **Подписки и уведомления:**  
  Можно выбрать интересы/каналы для подписки, включить/выключить уведомления.
  Автоматические рассылки новых событий по выбранным каналам Telegram.

- **Добавление событий организаторами:**  
  Через /add_event — организатор вводит данные события (название, дата, описание, ссылка).
  После модерации событие появляется в афише и рассылке.

- **Привязка профиля:**  
  Через /profile — связка Telegram-аккаунта с users платформы (через код подтверждения).

#### ЧАСТЫЕ ВОПРОСЫ

- **Как авторизоваться через Telegram?**  
  Введите /profile и следуйте инструкции.

- **Как добавить событие в календарь?**  
  После выбора события нажмите “В календарь”.

- **Можно ли добавить собственное событие?**  
  Да, через /add_event — после проверки оно появится на платформе.
