#### PUBLISHER: АВТОМАТИЧЕСКАЯ РАССЫЛКА СОБЫТИЙ

---

#### ОБЩЕЕ ОПИСАНИЕ

**kudab-publisher** — сервис автоматической рассылки новых и отобранных событий в Telegram-каналы и группы.

- Используется для регулярных публикаций анонсов в тематические и региональные каналы.
- Взаимодействует с API kudab.ru (получает список событий и расписание рассылки).
- Может работать как бот или как отдельный скрипт, поддерживает несколько каналов сразу.

---

#### СТЕК

- **Язык:** Python 3.11+
- **Библиотеки:** aiogram, httpx, apscheduler
- **Транспорт:** REST API (kudab-api), Telegram Bot API
- **Интеграция:** работает отдельно от основного бота, но использует единые токены/настройки

---

#### КОНФИГУРАЦИЯ И ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ

| Переменная             | Описание                                    |
|------------------------|---------------------------------------------|
| PUBLISHER_BOT_TOKEN    | Telegram bot token для публикации           |
| API_URL                | URL до backend API (например, http://kudab-api:9000/api/) |
| PUBLISH_CHANNEL_IDS    | Список channel_id для рассылки (через запятую) |
| SCHEDULE_CRON          | Расписание рассылки (например, каждый час)  |

---

#### СЦЕНАРИИ РАССЫЛКИ

- Периодически (или вручную) publisher обращается к API и получает список новых/актуальных событий.
- Формирует текст анонса и отправляет его во все подключённые Telegram-каналы.
- Поддерживает фильтрацию по интересам, гео, типу события — по настройкам канала.
- Ведёт логи успешных/неуспешных публикаций.
- Может отправлять “разовые” анонсы по webhook-запросу из API.

#### РОЛИ И ПРАВА

- Рассылкой управляет только publisher-бот (или админ через панель).
- Каналы и их настройки указываются в .env или через административный API.
- Логи доступны модератору/администратору через отдельную страницу или выгрузку.

---

#### ОПЫТ ПОЛЬЗОВАТЕЛЯ

- Пользователь подписывается на Telegram-канал kudab.ru (или тематические каналы).
- Получает только свежие, модерационные события (без дублей и спама).
- В каждом анонсе есть кнопки “Пойду”, “В избранное”, “В календарь” (через бот).
